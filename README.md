# -offer-Java-
这是我尝试使用java实现的在这本书中比较经典的算法

我发现网上很多实现剑指offer的帖子，都是直接用语言实现，讲解很少，且代码中没有注释。这样很不方便理解。
我对比了原版教材，发现作者在讲解每一块内容时，都做了详细的解释，但是配的源代码是c++的，对很多不熟悉c++的人来说并不是很友好
所以我决定开一个仓库，来记录一下自己的刷题历程，也方便大家参考一下，书中的很多经典算法的java实现。
